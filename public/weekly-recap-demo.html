About the Weekly Recap

The Weekly Recap popup shows users a summary of all the tasks they completed during the current week. It appears where the old "Notification subscription saved successfully!" message used to show.

Features:
  • Shows completed tasks from the current week (Sunday to Saturday)
  • Displays task titles and completion dates
  • Automatically appears weekly (configurable frequency)
  • Beautiful dark theme design matching the app
  • Auto-dismisses after 10 seconds
  • Manual close button

Implementation Details

The weekly recap system includes:

  • Backend API: /api/tasks/completed/week endpoint to fetch completed tasks
  • Frontend Script: weekly-recap-popup.js handles the popup logic
  • Local Storage: Tracks when the popup was last shown to avoid spam
  • Responsive Design: Works on both desktop and mobile devices
